# Support Page Draft

## Blackhole Support

Blackhole is a macOS black hole simulation app focused on gravitational lensing, accretion disk visuals, and cinematic space imagery.

## Requirements

- macOS 14 or later
- Metal-capable Mac recommended

## Troubleshooting

### The app feels slow

- Lower the computation mode
- Reduce the window size
- Use a lighter background image

### My custom background does not appear

- Re-select the file or folder from the app settings
- Make sure the selected file is an image format supported by macOS

### Visuals look different on another Mac

- Rendering quality depends on GPU capability and selected computation mode
- Intel Macs and lower-end GPUs may need lighter settings

## Privacy

Blackhole does not upload your selected background images or local files to a server.
